# Northcoders News API

## Getting Started

Install packages:

```bash
npm install
```

```bash
npm install dotenv
```

Please install PSQL for your MAC or PC

## Create the following files and add data to each respectively file:

./.env.test
```bash
PGDATABASE = nc_news_test;
```

./.env.dev
```bash
PGDATABASE = nc_news;
```

./.gitignore
```bash
.env.*
```
